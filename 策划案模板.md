## 一、游戏概述

### 1.1 核心概念
一款现代题材的第一人称恐怖生存游戏，玩家扮演鬼屋试睡员，需要在**午夜前收集防御道具**，在**午夜后抵御鬼怪的进攻**，坚持到凌晨 6 点即为胜利。

### 1.2 核心体验
- **紧张的双阶段循环**：午夜前（收集期）的紧迫感 + 午夜（防御期）的压迫感
- **资源管理**：道具有限，选择哪些道具直接影响生存概率
- **信息不对称**：通过监控/声音，以及发生的诡异事件判断鬼怪位置，做出实时决策
- **沉浸式恐怖**：现代场景，古典小别墅，熟悉的日常环境被异化

### 1.3 参考作品
- 《玩具熊的五夜后宫》（FNAF）系列——核心防御玩法
- 《恐鬼症》（Phasmophobia）——道具交互逻辑
- 《逃生》（Outlast）——第一人称视角与氛围营造

### 1.4 目标平台
PC（Windows），优先支持键鼠操作，可选支持手柄。

## 三、核心玩法：双阶段循环

### 3.1 整体时间线

```
22:00 ───── 23:59 ───── 00:00 ───── 06:00
│              │           │            │
入场        收集期      午夜         天明
                         ↓
                   鬼怪开始活跃
```

- **22:00 ~ 23:59（收集阶段）**：自由探索、收集道具、熟悉地图、布设防御
- **00:00 ~ 06:00（防御阶段）**：鬼怪进攻，玩家在监控室/安全点进行防御操作

### 3.2 第一阶段：午夜前收集（22:00 - 23:59）

#### 目标
在限定时间内尽可能多地收集关键道具，为午夜后的防御做准备。

#### 核心机制

| 机制 | 说明 |
|------|------|
| **限时探索** | 游戏内时间 2 小时（≈ 实际 5-10 分钟），到达 23:59 强制进入防御阶段 |
| **道具散布** | 道具随机分布在地图各处，每局位置不同，增加重复可玩性 |
| **体力系统** | 玩家有体力值，跑动消耗体力，体力耗尽只能慢走 |
| **初级灵异** | 收集阶段已有轻微灵异现象（灯光闪烁、门自动关闭、远处脚步声），但不致命 |
| **地图熟悉** | 玩家可在收集期记忆地图布局、出口位置、安全点位置 |

#### 可收集道具分类

| 类别 | 道具示例 | 用途 |
|------|---------|------|
| **防御类** | 圣水、盐、护身符、铁栅栏 | 驱赶/阻挡鬼怪 |
| **信息类** | 手电筒、监控摄像头、灵异探测器（EMF）、温度计 | 侦测鬼怪位置与类型 |
| **消耗类** | 电池、镇定剂（恢复精神力）、急救包 | 维持设备与状态 |
| **剧情类** | 日记残页、新闻剪报、死者遗物、录音带 | 解锁剧情与隐藏结局 |
| **陷阱类** | 捕鬼器、延迟符文、闪光弹 | 一次性格挡/削弱鬼怪 |

#### 收集阶段 UI 要素
- 倒计时器（显示当前游戏内时间）
- 道具栏（显示已收集道具，有容量上限）
- 体力条
- 小地图（可选，高难度禁用）

### 3.3 第二阶段：午夜后防御（00:00 - 06:00）

#### 目标
利用收集的道具，在固定位置（或可移动）抵御鬼怪进攻，存活到凌晨 6 点。

#### 核心机制

| 机制 | 说明 |
|------|------|
| **多路线进攻** | 鬼怪从多个入口/方向接近玩家，需切换监控查看 |
| **资源消耗** | 每次使用道具都是消耗性的，用完即无 |
| **耐力/理智值** | 长时间盯着鬼怪/处于黑暗中会降低理智值，理智归零产生幻觉甚至即死 |
| **音频线索** | 听脚步声、呼吸声、敲门声判断鬼怪距离，戴耳机体验最佳 |
| **时间压力** | 游戏内每小时 ≈ 实际 1-2 分钟，共 6 小时，越接近天亮鬼怪越凶猛 |

#### 玩家可进行的操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 切换监控画面 | 1-9 / 方向键 | 查看不同位置的摄像头 |
| 关门/封锁入口 | Space / 鼠标点击 | 消耗电力/道具，阻挡鬼怪 |
| 使用道具 | 右键 / E 键 | 对当前目标使用装备的道具 |
| 手电筒 | F | 照明，消耗电池 |
| 屏息/躲藏 | Ctrl | 暂时不被鬼怪发现，消耗体力 |

#### 鬼怪行为模式（示例）

| 鬼怪 | 特征 | 移动路线 | 弱点 |
|------|------|---------|------|
| **影人** | 人形黑影，行动缓慢但无声 | 走廊→通风管道→监控室 | 惧怕光照，手电筒照射可短暂驱退 |
| **灵火** | 蓝色火焰，移动快 | 沿电路/电线移动 | 惧怕圣水，可被盐圈阻挡 |
| **镜鬼** | 仅出现在镜面/屏幕反光中 | 跳跃式移动，出现在玩家背后 | 打碎镜子可暂时封锁其移动路线 |
| **噬忆者** | 最终夜 Boss，集合所有鬼怪特征 | 所有路线 | 需按特定顺序使用多种道具 |

#### 防御阶段 UI 要素
- 时间显示（当前游戏内时间）
- 监控画面（可切换）
- 道具快捷栏
- 电力/理智值/体力
- 威胁指示器（显示鬼怪大致方向和距离）
- 当前已封锁的门/入口状态

## 四、场景设计

### 4.1 场景选址方案（选其一）

| 场景 | 优点 | 适合的鬼怪类型 |
|------|------|-------------|
| **废弃写字楼** | 走廊多、房间多、监控摄像头逻辑通顺 | 办公室事故亡魂 |
| **旧医院** | 天然恐怖氛围、病房/走廊/地下室结构 | 病人/医生亡魂 |
| **废弃学校** | 教室/走廊/体育馆丰富空间、大众共鸣强 | 学生/教师亡魂 |
| **大型商场** | 多层结构、店铺多样、监控系统自然存在 | 火灾/事故亡魂 |
| **老旧公寓** | 紧凑空间、邻居式的恐惧感 | 住户亡魂 |

### 4.2 关键区域设计

| 区域 | 功能 |
|------|------|
| **监控室 / 安全点** | 玩家防御阶段的根据地，有监控屏幕、总电闸、道具箱 |
| **关键入口 × 3-5** | 鬼怪进攻的路线，玩家可封锁/监控这些入口 |
| **道具刷新点 × 10-15** | 收集阶段道具出现的可能位置 |
| **电源室** | 电力系统所在，鬼怪可能破坏，需玩家修复 |
| **隐藏房间** | 藏有剧情关键道具，需要特定条件解锁 |

## 五、美术风格

### 5.1 整体方向
- **写实风格**（PBR 材质，真实光照）
- 色调：收集阶段偏暖色（傍晚灯光），防御阶段偏冷色（月光/应急灯）
- 恐怖感来源：光影对比、颗粒效果、镜头晃动、UI 干扰（雪花屏、闪烁）

### 5.2 关键视觉元素
- 监控画面风格：低画质、灰度/夜视效果、画面撕裂
- 鬼怪设计：半透明、扭曲、非完整人形（避免过于直白的血腥）
- 环境叙事：血迹、散落的文件、损坏的家具、墙面涂鸦

## 六、音频设计

### 6.1 环境音
| 音效 | 说明 |
|------|------|
| 持续低鸣 | 背景氛围，低频嗡嗡声 |
| 风声/管道声 | 建筑自然声音，偶尔出现制造紧张 |
| 时钟滴答声 | 仅在收集阶段，提醒时间流逝 |
| 心跳声 | 鬼怪接近时渐强 |

### 6.2 交互音效
| 音效 | 说明 |
|------|------|
| 脚步声 | 不同地表材质不同音效 |
| 门的开/关/锁 | 关键反馈音效 |
| 监控切换 | 电流滋滋声 |
| 道具使用 | 各道具独特音效 |

### 6.3 鬼怪音效
| 音效 | 说明 |
|------|------|
| 呼吸声 | 鬼怪极近时，3D 空间定位 |
| 脚步声 | 判断鬼怪移动方向的关键线索 |
| 尖啸/低语 | 攻击前兆，给玩家反应时间 |
| 环境扭曲 | 鬼怪靠近时周围的音效变形 |

## 七、技术规格

### 7.1 引擎与工具
- **引擎**: Unity（版本见 `ProjectVersion.txt`）
- **语言**: C#
- **渲染管线**: 内置渲染管线 / URP（根据性能需求选择）
- **版本控制**: Git

### 7.2 核心系统架构

```
Assets/Script/
├── GameManager/          # 游戏主控、时间系统、昼夜切换
│   ├── GameManager.cs
│   ├── TimeManager.cs
│   └── PhaseController.cs    # 收集/防御阶段切换
├── Player/               # 玩家控制
│   ├── PlayerController.cs
│   ├── PlayerStatus.cs       # 体力、理智值、生命
│   └── Inventory.cs          # 道具栏系统
├── Enemy/                # 鬼怪
│   ├── EnemyBase.cs
│   ├── EnemyAI.cs            # 行为树/状态机
│   ├── EnemySpawner.cs
│   └── [具体鬼怪].cs
├── Items/                # 道具系统
│   ├── ItemBase.cs
│   ├── ItemSpawner.cs
│   └── [具体道具].cs
├── Interaction/          # 交互系统
│   ├── Interactable.cs
│   ├── DoorController.cs
│   └── MonitorSystem.cs
├── UI/                   # 界面
│   ├── HUDController.cs
│   ├── MonitorUI.cs
│   ├── InventoryUI.cs
│   └── PauseMenu.cs
└── Audio/                # 音频管理
    └── AudioManager.cs
```

### 7.3 性能目标
- 稳定 60 FPS（目标配置：GTX 1060 / RX 580）
- 最低配置 30 FPS（GTX 750 Ti / 集成显卡）
