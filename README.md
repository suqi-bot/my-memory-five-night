# 我的回忆 (My Memory)

类玩具熊的五夜后宫风格的 Unity 恐怖游戏。

## 游戏简介

玩家被困在一个充满恐怖回忆的场所中，需要利用有限的资源和策略来对抗夜晚的威胁，存活到天亮。

## 系统概述

### 核心系统
1. **音频系统** - 管理游戏音频，包括BGM、音效、UI音效和3D空间音效
2. **缓存池系统** - 管理游戏对象复用，优化性能
3. **事件中心系统** - 提供全局事件管理，实现模块间通信
4. **单例模式系统** - 提供全局唯一的管理器实例

### 游戏系统
5. **恐怖氛围系统** - 提供沉浸式恐怖体验，包括环境音效、视觉效果和恐怖事件
6. **玩家系统** - 管理玩家控制、移动、视角和交互
7. **场景物品系统** - 管理场景中的可交互物品

### 管理系统
8. **UI系统** - 管理用户界面，包括面板显示、隐藏和切换
9. **UI管理器系统** - 提供统一的UI管理接口
10. **游戏数据管理系统** - 管理游戏数据，支持数据存储和读取

### 辅助系统
11. **有限状态机系统** - 提供状态管理框架
12. **物品接口系统** - 定义物品交互标准接口
13. **启动脚本系统** - 游戏初始化和启动流程
14. **面前物品检测系统** - 检测玩家面前的可交互物品

## 快速开始

### 1. 运行游戏
1. 使用Unity打开项目
2. 打开场景 `Assets/Scenes/`
3. 点击运行按钮

### 2. 开发指南
1. 阅读 `Docs/INDEX.md` 了解系统文档
2. 根据需要查看具体系统文档
3. 参考示例代码进行开发

### 3. 系统集成
1. 使用事件中心系统进行模块通信
2. 使用单例模式系统管理全局状态
3. 使用UI系统管理界面

## 项目特色

### 恐怖氛围系统
- 环境音效系统（低鸣、风声、心跳）
- 视觉效果系统（镜头晃动、屏幕闪烁、噪点）
- 恐怖事件系统（灯光闪烁、门自动关闭、脚步声）

- 3D空间音效
- 循环音效管理
- 音量控制
