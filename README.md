# 我的回忆 (My Memory)

类玩具熊的五夜后宫风格的 Unity 恐怖游戏。

## 游戏简介

玩家被困在一个充满恐怖回忆的场所中，需要利用有限的资源和策略来对抗夜晚的威胁，存活到天亮。

## 项目结构

- `Assets/Script/` - 游戏核心脚本
  - `Player/` - 玩家控制与状态管理
  - `SceneItem/` - 场景物品交互
  - `UI/` - 界面管理
- `Assets/Scenes/` - 游戏场景
- `Assets/Resources/` - 资源文件

## 运行环境

- Unity 版本: 请查看 `ProjectSettings/ProjectVersion.txt`
- 平台: Windows
